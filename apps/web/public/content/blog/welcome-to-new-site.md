# Hello, World!

古川(Hatake)の新しいポートフォリオサイトへようこそ。
このサイトは **Furukawa Archive OS** として、私の研究活動や制作物をアーカイブするために作られました。

## 新機能

- **Admin Console**: ブラウザから直接コンテンツを編集できるようになりました。
- **Real-time Updates**: 訪問者数やおやつ代の通知がリアルタイムで届きます。
- **Blog**: このようにMarkdownでブログを書くことができます。

これからも活動を更新していくので、ぜひチェックしてください！
